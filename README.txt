
📘 ДОКУМЕНТАЦІЯ — Telegram Bot для гри з балансом та панеллю адміністратора

🔧 ОПИС
Цей бот створений на Python з використанням бібліотеки telebot. 
Він дозволяє користувачам грати у демо та реальні ігри, поповнювати баланс, 
та надає адміністраторам інструменти для керування користувачами.

📂 СТРУКТУРА ПРОЄКТУ
- bot.py — основна логіка команд Telegram
- game.py — механіка гри (раунди, учасники, переможці)
- model.py — моделі бази даних (User, Versamento)
- conectdb_Roman.py — підключення до бази даних
- .env — зберігає токен бота

👤 КОМАНДИ КОРИСТУВАЧІВ
/start — реєстрація користувача
/registration — введення email
/balence — перегляд балансу (реального та демо)
/versademo <сума> — участь у демо-грі
/versareal <сума> — участь у реальній грі

💼 КОМАНДИ АДМІНІСТРАТОРА
/adminpanel — панель адміністратора (статистика, статус гри, користувачі)
/AddCredit @username <сума> — додати кредити користувачу
/rimuve @username — видалити користувача
/blocca @username — заблокувати користувача

🧩 ДОДАТКОВО
/buyHivepoint — зарезервована команда для майбутніх функцій

📊 ІНФОРМАЦІЯ В PANЕL ADMIN
- Кількість зареєстрованих користувачів
- Загальний баланс реальних і демо кредитів
- Статус поточних ігор (активна / неактивна)
- Кількість гравців у кожному режимі

🧠 ЛОГІКА РОБОТИ
Користувач виконує команду → бот перевіряє у базі → виконує дію (створення, участь, оновлення балансу). 
Адміністратор може контролювати баланс, блокувати або видаляти користувачів.

📦 ЗАПУСК
1. Створіть файл .env з вашим токеном:
   TOKEN=1234567890:ABCDEFG...
2. Запустіть бота:
   python bot.py

✅ Поради
- Перейменуйте /balence у /balance
- Перейменуйте /rimuve у /rimuovi
- Додайте перевірку is_blocked у game.py
- Зробіть логування дій адміністратора
