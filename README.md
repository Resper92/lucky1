# 🎮 Telegram Bot для Гри

Цей бот, створений на Python, дозволяє користувачам брати участь у демо та реальних іграх, керувати своїм балансом та надає адміністраторам інструменти для управління.

---

## 🚀 Команди для Користувачів

| Команда | Опис |
| :--- | :--- |
| `/start` | Реєструє нового користувача в системі, надаючи початковий демо-баланс. |
| `/registration` | Запускає процес введення email для зареєстрованого користувача. |
| `/balence` | Показує поточний реальний та демо-баланс. |
| `/startdemo <сума>` | Дозволяє взяти участь у демо-грі, зробивши ставку з демо-балансу. |
| `/real <сума>` | Дозволяє взяти участь у реальній грі, зробивши ставку з реального балансу. |
| `/pay` | Запускає процес поповнення реального балансу через створення платіжного рахунку. |

---

## 🔐 Команди для Адміністраторів

| Команда | Опис |
| :--- | :--- |
| `/adminpanel` | Відображає панель адміністратора зі статистикою (кількість користувачів, загальні баланси) та статусом активних ігор. |
| `/AddCredit @username <сума>` | Додає вказану суму кредитів на реальний баланс користувача. |
| `/rimuve @username` | Видаляє користувача із системи та бази даних. |
| `/blocca @username` | Блокує користувача, забороняючи йому взаємодію з ботом. |

---

## 🔧 Запуск Бота

1.  **Створіть файл `.env`** з токеном вашого бота:
    ```
    TOKEN=1234567890:ABCDEFGHIJKL...
    ```
2.  **Встановіть залежності:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Запустіть бота:**
    ```bash
    python main.py
    ```
